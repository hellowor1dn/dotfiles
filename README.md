## Setup

1. Install XCode

2. Clone dotfiles:

```bash
git clone -b develop https://github.com/hellowor1dn/dotfiles
cd dotfiles
git submodule update --init --recursive
```

3. Run install

```bash
./install.sh
```

4. Reboot
